Поддерживаемые операционные системы
***************************************

       + Windows 98SE, 2000, XP, Vista, 7, 8.x (x86 & x64 Windows platforms)
 

Установка
***************

       2- Запустите файл "VCP_V1.4.0_Setup.exe" 
	   
	   3- Перейдите в каталог установки, - например, C:\Program Files (x86)\STMicroelectronics\Software\Virtual comport driver
	   
	   4- Зайдите в каталог, соответствующий вашей ОС Win7 или Win8
			+ Затем :
				- Запустите dpinst_x86.exe, если вы используете 32-битную версию ОС
				- Запустите dpinst_amd64.exe, если вы используете 64-битную версию ОС
			+ Следуйте инструкциям.
